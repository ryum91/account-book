## History ##
	* 가계부 내역
	* Date - 날짜
	* Type - 0:수입 / 1:지출
	* AccountIdx - 계좌 인덱스
	* CategoryIdx - 카테고리 인덱스
	* Title - 제목
	* Message - 메시지	
	* Price - 가격

## Category ##
	* 카테고리
	* Idx - 인덱스
	* Type - 0:수입 / 1:지출
	* Name - 이름
	* ParentIdx - 부모 카테고리 인덱스

## Account ##
	* 결제 수단
	* Idx - 인덱스
	* Name - 이름
	* Color - 색상
	* InitialBalance - 초기 잔액
